# 支行日用工具箱

GitHub 仓库：`xyyyp1314/-`

这是一个用于支行日常测算的小程序工具箱，制作者为“巩义支行个金部”。

## 当前包含的工具

- `大额订单`：单笔测算转让价格、后手利率、保留利率、参考公允价
- `日均计算器`：测算当前完成度、日均缺口、平线、倒推 A*
- `大额订单套利`：反推挂单日期、隐藏收益、重挂空间、市场对比
- `临界客户提升`：按月日均 / 年日均倒推后续平均值与需要补的资产

## 本地运行

1. 使用微信开发者工具打开项目根目录
2. 选择小程序项目并导入
3. 编译后即可本地预览

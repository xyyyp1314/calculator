# 支行日用工具箱

GitHub 仓库：`xyyyp1314/-`

“支行日用工具箱”是一个面向支行日常经营测算场景的微信小程序，制作者为“巩义支行个金部”。项目把存单收益测算、套利盯盘、日均推进、临界客户提升和存款日平线等常用计算工具集中到一个入口里，适合内部日常使用和快速测算。

## 项目定位

- 面向支行个金条线的日常经营测算
- 以“小程序内直接输入、直接出结论”为核心体验
- 以中文业务口径为主，尽量减少公式理解门槛
- 当前版本为纯前端本地运行，无后端依赖

## 当前功能

### 1. 大额订单

用于单笔大额可转让存单收益测算，支持多种反推方式：

- 按转让价格反推后手显示利率、保留利率、今日年化
- 按后手显示利率反推转让价格、保留利率、今日年化
- 按保留利率反推转让价格、后手显示利率、今日年化
- 按当前执行利率给出参考公允价、利差、套利空间

### 2. 日均计算器

用于日常日均推进测算：

- 看当前日均新增、时点新增、完成度
- 看距离目标日还差多少日均
- 自动计算剩余平线总量、剩余平线增量
- 支持填入平线增量后倒推 `A*` 和 `A*/A`

### 3. 大额订单套利

用于盯盘大额可转让存单的套利空间：

- 根据页面价格和页面后手利率，反推大概率挂出日期
- 估算“老利率”与今天真实收益口径之间的偏差
- 测算隐藏收益、同保留利率重挂价、自定义重挂价
- 按当前市场利率口径做参考价和价差对比

### 4. 临界客户提升

用于冲刺月日均 / 年日均目标：

- 支持“月日均”和“年日均”两套口径
- 自动根据当前统计日和目标统计日计算剩余天数
- 直接告诉使用者后几天平均要做到多少
- 直接给出“还要补多少资产”的结果

### 5. 存款日平线

用于机构 / 同业存款日平线测算：

- 支持按月份自动生成考核节点
- 按“周五 ×1、月末 ×3”规则计算加权平均
- 正向测算日平线结果
- 输入目标日平线后，自动倒推出各节点平均需要达到的余额
- 已填部分节点时，支持过程推算剩余节点需要保持的水平

### 6. 登录与设备校验

项目带有首次验证登录流程：

- 首次进入先进行使用者验证
- 验证通过后，当前设备本地记住登录状态
- 后续可直接进入工具箱
- 当前校验方式为前端本地校验，适合内部演示和临时使用

## 页面结构

当前页面结构如下：

- `pages/auth/index`
  - 首次验证页
- `pages/home/index`
  - 工具箱首页
- `pages/scanner/index`
  - 日均计算器
- `pages/calculator/index`
  - 大额订单
- `pages/arbitrage/index`
  - 大额订单套利
- `pages/critical/index`
  - 临界客户提升
- `pages/depositline/index`
  - 存款日平线

说明：

- `首页 / 日均 / 订单 / 套利 / 临界` 作为 tab 页面存在
- `存款日平线` 当前通过首页入口进入，不在 tabBar 中单独展示

## 运行方式

1. 使用微信开发者工具打开项目根目录
2. 导入小程序项目
3. 编译后即可本地预览

## 数据与存储

当前版本不依赖后端接口，页面输入数据主要保存在本地缓存中，便于下次继续使用。不同工具页会使用各自的本地存储键保存最近一次输入状态。
